# Changelog

## 0.1.0 - 2026-07-21

- 完成当前小洛宝项目的 Part、pivot、时间轴、关键帧、预览和统一撤销编辑闭环。
- 支持版本化项目目录、最近项目、另存为、自动 recovery 和固定目标的安全生产发布。
- 保存前保留最多 5 份项目备份；恢复旧版本前自动备份当前版本。
- 明确仅支持 project/rig/motions schema v1，并在不兼容时阻止打开、保存和发布。
- 增加脱敏本地诊断导出、Windows 单实例、便携构建和 NSIS 安装包入口。
- 固定 SVGCanvas 输入并增加供应链、bundle 边界和正式素材性能检查。
- canonical rig/motions 导出改用单次原生目录选择器，并在日志中显示实际输出目录。
- 修复无路径的 Rust 结构化错误显示为 `[object Object]`，发布失败现在会显示具体错误码和阶段。
