## 3. 共享核心 `@ltypet/character-motion`

### 安装基线

```
packages/character-motion/
  ajv 8.20.0
  typescript 5.8.3
  vitest 3.2.7
```

零 React/DOM/svgcanvas/Tauri/矩阵库/缓动库依赖。

### 模块清单

| 模块 | 文件数 | 功能 |
|---|---|---|
| 类型定义 | `types.ts` | AffineMatrix, CharacterRigV1, MotionLibraryV1, MotionClipV1 等 |

| 结构校验 | `validateStructure.ts` | Ajv 编译 validator，stable issue codes |
| 语义校验 | `validateSemantics.ts` | 重复/环/引用不存在/矩阵奇异检测 |
| 仿射矩阵 | `affine2d.ts` | identity/multiply/invert/translate/rotate/scale/transformPoint/composeAroundPivot |
| Pivot | `pivot.ts` | computePivotInPartLocal |
| 缓动 | `easing.ts` | linear/easeIn/easeOut/easeInOut/cubicBezier (Newton-Raphson + bisection) |
| 帧时间 | `frameTime.ts` | frameToTime/timeToFrame/wrapFrame |
| 属性采样 | `sampleProperty.ts` | 稀疏关键帧逐属性采样 |
| Clip 采样 | `sampleClip.ts` | sampleMotionClip |
| 依赖图 | `dependencyGraph.ts` | topologicalOrder with cycle detection |
| 姿态解析 | `resolvePose.ts` | resolveWorldPose |
| 序列化 | `canonicalize.ts` + `fingerprint.ts` | 稳定字节输出 + SHA-256 |

## 6. 已知边界

1. **编辑器构建 1.46 MB**：包含 svgcanvas + 共享核心，chunk 警告保留到 P2
2. **Rig 为伪 rig**：编辑器使用 `buildPseudoRig()` 从 import data 动态构建，所有 Part 的 `logicalParentId` 和 `defaultRenderSlot` 为默认值；正式 rig 需从导入导出流程生成
3. **Procedural channel 未在编辑器中暴露**：`suppressProceduralChannels` 在 motions v1 schema 中定义但编辑 UI 未提供开关
4. **浏览器验证未执行**：pivot 旋转效果和 v1 导出/重开需在 `http://localhost:1430` 确认
